Kaartanimaties

Hier vindt u mijn kaartanimatie voor op de voorpagina, we moeten hier nog gepaste achtergronden voor zoeken.
Ik heb alle classes en ids aangepast naar de gepaste WordPress tags om ze compatibel te maken met de juiste voorpagina.
De achtergrond blijft in dezelfde positie terwijl de kaart omhoog beweegt, dit is niet mogelijk met gewone CSS code dus moest ik gebruikt maken van pseudo elementen (::after & ::before)

Werkwijze

Mark Kelly, de CEO van mijn stagebedrijf heeft ons laten weten dat we kunnen werken met Elementor Pro. Dit is een betalende plugin voor WordPress om designs extra snel te maken voor zijn WordPress website.
Maar toen ik gebruik maakte van dit programma kwam ik er al snel achter dat het helemaal niet goed werkte (gewoon Elementor Pro openen heeft de volledige layout van de front page verpest)
Ik heb hem dit laten weten en hij heeft dan gezegd dat we dit ook op onze manier mochten doen, zolang we het dan implementeren in WordPress. Dit is dus hoe ik te werk ga op mijn stageplek.
